# Water Billing System

A simple Water Billing System for managing household water usage and payments.

## Features

- Register customers
- Record water meter readings
- Calculate water bills
- Record customer payments
- View billing records

## Technologies

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Installation

1. Clone this repository.
2. Copy the project to your web server (e.g., XAMPP `htdocs`).
3. Import the database into MySQL.
4. Update the database connection settings.
5. Run the project in your browser.
